# Асинхронный вариант
Время поиска 4 монет<br>
Кол-во процессов - время выполнения:<br>
1 процесс - 48449 мс<br>
2 процесса - 43311 мс<br>
3 процесса - 23675 мс<br>
4 процесса - 22012<br>
8 процессов - 47994<br>
Можно заметить, с увеличением кол-ва процессов, время поиска уменьшается. Однако, при использовании процессов, превосходящих колличество ядер на устройстве не ускоряет работу.<br>
